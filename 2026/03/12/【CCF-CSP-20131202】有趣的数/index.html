有趣的数

时间限制： 1.0 秒

空间限制： 256 MB

题目描述

我们把一个数称为有趣的，当且仅当：

  1. 它的数字只包含 0,1,2,3且这四个数字都出现过至少一次。
  2. 所有的 0 都出现在所有的 1 之前，而所有的 2 都出现在所有的 3 之前。
  3. 最高位数字不为 00。

因此，符合我们定义的最小的有趣的数是 2013。除此之外，4 位的有趣的数还有两个：2031 和 2301。

请计算恰好有 n 位的有趣的数的个数。由于答案可能非常大，只需要输出答案除以 10的9次方+7的余数。

输入格式

从标准输入读入数据。

输入的第一行包含一个正整数 TT，表示共有 TT 组数据。

接下来 TT 行，每组数据占一行，分别包含一个正整数 n。

输出格式

输出到标准输出。

共输出 TT 行，每行一个整数，对应询问所求的答案，为恰好 n 位整数中有趣的数的个数除以 10的9次方+7 的余数。

输入数据 1

输出数据 1

数据范围

对于所有数据，1≤T≤100, 1≤n≤10001≤T≤100, 1≤n≤1000。

答案

解析

🧩 第一步：理解题目规则（搭积木的规则）

题目说“有趣数”要满足三条规则，我们用白话翻译一下：

  1. 数字必须由{0,1,2,3}组成，且四个数字都要出现至少一次。
    → 相当于你手里有四种颜色的积木（0红、1黄、2蓝、3绿），最后搭出来的塔必须四种颜色都有。
  2. 所有的0都在所有的1之前；所有的2都在所有的3之前。
    → 红色的0积木必须全部放在黄色的1积木前面；蓝色的2积木必须全部放在绿色的3积木前面。
    也就是说，一旦你放了1（黄色），后面就不能再放0（红色）；一旦你放了3（绿色），后面就不能再放2（蓝色）。
  3. 最高位（第一位）不能是0。
    → 塔的底座不能是红色积木。

🧱 第二步：首位为什么必须是2？

我们先想想第一位能放什么：

  • 不能是0（规则3禁止）
  • 如果是1：那么0必须在1之前，但第一位之前没有位置，所以0没地方放 → 排除
  • 如果是3：那么2必须在3之前，同样2没地方放 → 排除
  • 如果是2：可以！2后面可以放0或3，都不违反规则。

所以第一位只能是2 。
这就好比：我们建塔的底座必须是蓝色的2积木。

🏗️ 第三步：搭积木的过程——6种中间状态

现在我们要从第一位（2）开始，一位一位往后面添加数字。每添加一位，我们手里的积木种类会发生变化。根据已经用了哪几种颜色，我们可以把“搭到一半的塔”分成6种状态：

状态编号 已经用过的数字 通俗理解（塔的样子） 还没用的数字
0 {2} 只有蓝色 红、黄、绿
1 {2, 0} 蓝和红 黄、绿
2 {2, 3} 蓝和绿 红、黄
3 {2, 0, 1} 蓝、红、黄 绿
4 {2, 0, 3} 蓝、红、绿 黄
5 {2, 0, 1, 3} 四种颜色全齐了 无

注意：状态5就是最终的有趣数（四个数字都出现了）。

🔄 第四步：如何从一种状态变成下一种？（状态转移）

这是最关键的一步！我们想象在塔的末尾再放一块积木，看看能从哪些状态来，能变成哪些状态。

状态0（只有2）

  • 末尾可以放2（还是只有2，保持状态0）
  • 末尾可以放0 → 变成状态1（有了2和0）
  • 末尾可以放3 → 变成状态2（有了2和3）
  • 不能放1（因为1必须在0之后，0还没出现）

状态1（有2和0）

  • 末尾可以放2（保持状态1）
  • 末尾可以放0（保持状态1）
  • 末尾可以放1 → 变成状态3（有了2,0,1）
  • 末尾可以放3 → 变成状态4（有了2,0,3）

状态2（有2和3）

  • 末尾可以放3（保持状态2）
  • 末尾可以放0 → 变成状态4（有了2,3,0）
  • 不能放2（因为3已经出现，2不能再出现）
  • 不能放1（1需要在0之后，0还没出现）

状态3（有2,0,1）

  • 末尾可以放1（保持状态3）
  • 末尾可以放2（保持状态3，因为2可以重复出现）
  • 末尾可以放3 → 变成状态5（四色全）
  • 不能放0（0必须在1之前，现在1已经出现）

状态4（有2,0,3）

  • 末尾可以放0（保持状态4）
  • 末尾可以放3（保持状态4）
  • 末尾可以放1 → 变成状态5（四色全）
  • 不能放2（3已经出现）

状态5（四色全）

  • 末尾可以放1或3（保持状态5）
  • 不能放0或2（0不能在1之后，2不能在3之后）

📊 第五步：把转移关系变成数学公式

现在我们用 dp[i][j] 表示：长度为 i 的数字，且处于状态 j 的个数。

根据上面的转移关系，我们可以写出递推公式：

当前状态 可以从哪些前驱状态来 转移方式 公式
0 只能是前一位状态0 末尾加2 dp[i][0] = 1
1 前一位状态0 或 状态1 状态0加0，状态1加0或2 dp[i][1] = dp[i-1][0] + 2*dp[i-1][1]
2 前一位状态0 或 状态2 状态0加3，状态2加3 dp[i][2] = dp[i-1][0] + dp[i-1][2]
3 前一位状态1 或 状态3 状态1加1，状态3加1或2 dp[i][3] = dp[i-1][1] + 2*dp[i-1][3]
4 前一位状态1 或 状态2 或 状态4 状态1加3，状态2加0，状态4加0或3 dp[i][4] = dp[i-1][1] + dp[i-1][2] + 2*dp[i-1][4]
5 前一位状态3 或 状态4 或 状态5 状态3加3，状态4加1，状态5加1或3 dp[i][5] = dp[i-1][3] + dp[i-1][4] + 2*dp[i-1][5]

一些语法

vector<vector >dp(maxx + 10, vector(6,0));

  • vector<vector<ll>>：声明了一个向量的向量，也就是二维数组。外层向量的每个元素本身又是一个vector<ll>（存放long long类型的一维向量）。
  • dp：变量名。
  • (max_n + 1, vector<ll>(6, 0))：这是构造函数的参数，用来初始化这个二维向量。
    • 第一个参数 max_n + 1：指定外层向量有多少个元素。这里我们想创建 max_n + 1 行（对应长度从 0 到 max_n）。
    • 第二个参数 vector<ll>(6, 0)：指定每个外层元素（即每一行）的初始值。这里创建了一个内层向量，它有 6 个元素，每个元素初始化为 0。也就是说，每一行都是一个长度为 6 的向量，且所有值都是 0。
